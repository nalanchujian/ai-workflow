# 需求定义输出模板

写入 `<tasks-root>/<task-id>/requirement-analysis.md`。聊天只展示当前疑点；全部疑点处理完后再摘要结果和文件路径。

```yaml
---
task_id: <task-id>
node: requirement-intake
status: completed | blocked
attempt: 1
template_version: 2
---
```
