**测试数据**

- 权限矩阵：Tracking/Free trial × View details/Export 有权与无权；准备方 QA
- 指标配置：白名单内 4 项、白名单外指标及不同顺序；准备方 QA
- 指标边界：缺失、0、正数、部分缺失、全部缺失及分母 0；准备方数据/API Owner
- 趋势数据：日/周粒度、不同后端起止时间和空数据；准备方数据/API Owner
- Fan 数据：至少两页、七列顺序不同、有效 profile link；准备方数据/API Owner
- 导出数据：All/Selected creator、All/Selected data、gross/net、成功和失败响应；准备方 QA 与 API Owner
